> 东北戏剧🥹。

张庭秀未曾说话深搭一躬，口尊声王二小姐你要细听  
休当我是花儿乞丐，我本是你的二哥转回家中  
问一声岳父岳母二老人家可都好，恩妹你安宁不安宁  
...  
二妹稳坐观花亭，听二哥从头至尾表表往情  
想当年我家不在你们苏州住，洪桐县里有门庭  
只因为洪桐遭荒旱，一连三年没有收成  
头年荒旱不下雨，第二年五月端阳起蝗虫  
第三年三月十三下场透雨，直下到八月十五才把天晴  
旱的旱来涝的涝，黎民百姓受贫穷  
我的父一见日子过不下去，领我们全家逃荒来到苏州城  
我父推着跨车子走，我母拉纤一根绳  
二哥手中一个柳罐斗，啥也没有溜溜空  
...  
身上无钱难住店，在你们家堂庙里把身容  
我父张全是木匠，锛凿斧锯挂出山门中  
正是张家时来运转，赶上王府修大工  
你父说本城木匠一个也不用，外来的手艺人不旷工  
...  
我父在前厅掌尺画线，二哥身小力薄做个软木工  
在后院给二妹刻一个镜子架，刻的是凤凰落梧桐  
监工名叫王三老，这老头人好心眼更公平  
当你父面前常把我夸奖，说我手巧心也灵  
他说我头上麻子不叫麻子，朝王散，背膀斜带两条龙  
到日后一根玉带不够二哥系，两根玉带拧成绳  
你父相信三老话，认下我义子作螟蛉  
...  
你父怕树粗缰短拴不住马，河宽水浅难养龙  
先为螟蛉后招婿，许下二妹你婚事一宗  
自那日不把木工做，送到南学把书攻  
二哥我念书念到龙虎日，嘉靖皇爷开考棚  
进京科考不是一人去，赵囊姐夫同伴行  
赵囊拉马前院走，二哥拉马后花园中  
打发春红去送信，二妹你隔着竹帘与我饯行  
我拿花枝小扇作表记，你给我传家之宝白玉盅  
你言说人若回来表记也得在，无杯对面不相逢  
…  
依我要从旱路走，赵囊偏要水路行  
搭个跳板官船上，七日七夜直奔京城  
那一天风狂雨大船不稳，谁知赵囊把歹计生  
图宝害人下毒手，将我推落大江中  
盗去银钱还不算，最伤心传家之宝白玉金盅  
我得下气恼伤寒病，一场大病可不轻  
三场皆过误了考，手中没剩半文铜  
流落街头无处去，花子堆里拜弟兄  
...  
赵囊献宝得官回家转，言说二哥我命丧南京  
他是你王家乘龙婿，我怎敢再把书信通  
他若知道我在世，接连会把毒计生  
隐姓埋名度岁月，难舍恩妹王兰英  
沿路来到苏州地，来给二妹把信通  
张王李赵你随意嫁，别跟我花子受贫穷  
我一天常挨半天饿，有张狗皮能过冬  
二妹要别的身没有，头上是个铁证凭  
相面叫它朝王散，二哥叫它麻子坑  
开花大帽挺几挺，你看我是不是二哥回家中  

听二哥在京城连遭不幸，不由我兰英热泪洒前胸  
二哥在外受贫苦，家中也是祸不单行  
赵囊图宝害命还家转，花言巧语将我父蒙  
说的是二哥已死他尽孝，又养老来又送终  
...  
赵囊到县衙去告状，雪花白银把贿赂行  
常言清酒红人面，清官买一个糊涂虫  
从南京捎一根毛竹板，一头重来一头轻  
五十板子换上人一位，你父挺刑不过认下杀人凶  
...  
夜晚来到你们张家院，一场大火烧得多苦情  
...  
二哥呀，你今天回来真凑巧，晚回一天不相逢  
苏大公子下聘礼，明天来娶王兰英  
他的万贯家财我不要，走马章台无真情  
...  
大街以上纷纷嚷，说京城来一个巡按公  
十里长街扎公馆，岁数不大堂口清  
你何不拦轿去告状，我教你被告人几名  
头状你告我父老王宪，一女二聘罪不轻  
二状你把知县告，做官贪赃法不容  
三状你把赵囊告，他害你张家太苦情  
四状你把二妹告，我到大堂去证明  
你是去来是不去，耷拉个脑袋你咋不吱声  
...  
闻听二妹说一遍，字字句句打动我心中  
...  
要提起修宅建院我都会，提起打官司告状可不中  
我从小得个怕官病，一到大堂就发蒙  
大老爷一拍惊堂木，吓得我小腿转筋生生疼  
...  
二哥说出泄气话，你可气坏王兰英  
害父之仇你不报，夺妻之恨你能容  
有道人穷志不短，谁像你人穷志也穷  
从前看你像竹竿样，长来长去节节空  
从前看你像豆芽菜，长来长去弯了弓  
车辙沟泥鳅来回跑，跑到多远难成龙  
常言好虎一个能拦路，一百个黑瞎子五十对熊  
二姐我越骂越生气  
...  
（张庭秀越听我咋越爱听）  
你说我熊我就熊，我可不受你糊弄  
谁不知民告官有杀头罪，一到大堂活不成  
大堂打死张庭秀，你好嫁给苏大相公  
二哥我生来就是穷苦命，乞丐离不开花子棚  
再不离开王家府，想要逃走万不能  
张庭秀假意往外走  
...  
一旁急坏王兰英，二哥二哥慢点走  
你怎忍心把二妹扔  
二妹若有害你意，何必等你六年功  
气头说几句伤人话，原谅二妹我年纪还轻  
...  
往外走，往回拽，嘎巴拉断稻草绳  
...  
张庭秀只觉怀里咋这么空  
...  
乔装改扮访二妹，又装窝囊又耍熊  
今天我不给他皇家印，看他还有啥章程  

张庭秀走上近前深施一礼，口尊二妹你要听  
方才咱俩在这来讲话，你捡没捡到二哥的物一宗  
...  
问二哥丢了什么物  
...  
四方方一块老黄铜  
...  
问二哥黄铜在哪得  
...  
在京城要饭六年功，积攒下这块老黄铜  
...  
问二哥黄铜有何用  
...  
要饭吃凉一口热一口，揣在怀里肚子不疼  
...  
二妹我也有肚子疼的病，送给我揣揣中不中  
...  
这黄铜光管男来不管女，女的要揣上肚子拧劲疼  
...  
你把黄铜送给我，打一副手镯中不中  
...  
你们家金子银子装满柜，为啥偏要这块生铜  
打一锤子四下崩，宁死我也不舍这份人情  
...  
莫非黄铜是无价宝  
...  
它跟二哥我连着性命  
...  
你不给我不要，咱俩谁也要不成  
手拿黄铜往外走，浇花井前把身停  
二姐我假意要撒手  
...  
哎呀不好，吓坏八府巡按公  
别撒手别撒手，你要撒手把我坑  
我说黄铜本是假，它是皇家印一封  
真情实话告诉你，愿意扔来你就扔吧  
...  
二姐闻听心欢喜，让我扔来我偏不扔  
你当我不识皇家印，我父也做过四品卿  
问二哥所得官几品  
...  
小小八府巡按公  
...  
问二哥为何扮作花子样  
...  
来访恩妹二兰英  
...  
问二哥访的怎么样  
...  
掌印夫人你当成  
...  
二妹恩情如山重，敬佩你在苏州等我六年功  
...
